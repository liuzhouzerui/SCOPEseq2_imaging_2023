# scopeseq-v2

# Installation
### Environment & Dependencies
This pipline requires Python >= 3.6 and the packages:
- scipy >= 1.1
- numpy
- pandas

### Installing the code 
Once you have set up the environment, clone this repository and install.
```
git clone git@github.com:simslab/scopeseqv2_2021.git
cd scopeseqv2_2021
pip install .
```
